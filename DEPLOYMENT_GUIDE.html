01
Get Your TMDB API Key
Free · Takes 2 minutes

CineSearch uses The Movie Database (TMDB) API, which is completely free for personal projects.

🔑 Step-by-step API Key Setup
  • 1
    Go to themoviedb.org/signup and create a free account
  • 2
    Navigate to Settings → API in your account dashboard
  • 3
    Click "Create" and choose "Developer" for the type
  • 4
    Fill out the form (app name: "CineSearch", use: "Personal Project") and submit
  • 5
    Copy your API Key (v3 auth) — it looks like: a1b2c3d4e5f6...
02
Local Development Setup
Prerequisites: Node.js 16+ and npm

Install Dependencies

# Navigate to the cinesearch folder
cd cinesearch

# Install all packages
npm install

Configure Environment Variables

Copy the example env file and add your API key:

cp .env.example .env

Then open .env and replace the placeholder:

REACT_APP_TMDB_API_KEY=your_actual_api_key_here
⚠️
Never commit your .env file. It's already in .gitignore, but double-check before pushing to GitHub. Your API key must start with REACT_APP_ for Create React App to expose it to the browser.

Start the Development Server

npm start
✅
The app opens at http://localhost:3000. Hot reloading is enabled — changes appear instantly.

03
Push to GitHub
Required for Vercel deployment

Vercel deploys directly from a GitHub repository. Create a new repo and push your code:

# Initialize git (if not already done)
git init
git add .
git commit -m "Initial commit: CineSearch app"

# Create a new repo on github.com, then:
git remote add origin https://github.com/your-username/cinesearch.git
git branch -M main
git push -u origin main
🔒
Before pushing, confirm .env appears in your .gitignore. Run git status — if .env shows up as a tracked file, run git rm --cached .env first.
04
Deploy on Vercel
Free tier · Automatic HTTPS · CI/CD
🚀 Deployment Steps
  • 1
    Go to vercel.com and sign up / log in with your GitHub account
  • 2
    Click "Add New → Project" and import your cinesearch repository
  • 3
    Vercel auto-detects Create React App. Leave framework preset as "Create React App"
  • 4
    Expand "Environment Variables" and add your key:

Add Environment Variable in Vercel Dashboard

Name:   REACT_APP_TMDB_API_KEY
Value:  your_actual_tmdb_api_key
Environment: Production, Preview, Development
Build Settings (auto-detected — no changes needed)
Framework Preset:  Create React App
Build Command:     npm run build
Output Directory:  build
Install Command:   npm install

Click Deploy. Vercel will build and deploy your app. In ~2 minutes you'll get a live URL like:

https://cinesearch-yourname.vercel.app
✅
Every git push to main automatically redeploys. Vercel watches your repo and triggers a new build on every push. Pull request previews also get their own unique URL.
05
Custom Domain (Optional)
Point your own domain to Vercel

To use a custom domain like cinesearch.com:

  • 1
    In Vercel dashboard → your project → Settings → Domains
  • 2
    Add your domain name and follow the DNS configuration instructions
  • 3
    Add a CNAME record pointing to cname.vercel-dns.com in your DNS provider
  • 4
    Vercel automatically provisions a free SSL certificate via Let's Encrypt

Customization Guide
Make it your own

Change the Color Scheme

All colors are CSS variables defined in src/styles/GlobalStyle.js:

// Swap --accent to change the entire app's highlight color
--accent: #e8c547;    /* Gold → try #e84888 for pink */
--bg: #0a0a0f;         /* Dark bg → try #f5f5f0 for light mode */
--bg-card: #111118;   /* Card bg */

Add New API Endpoints

All TMDB calls live in src/utils/tmdb.js. Add new functions following this pattern:

// Example: Fetch movies by decade
export const getMoviesByDecade = (decade, page = 1) =>
  fetcher(buildUrl('/discover/movie', {
    primary_release_date.gte: `${decade}-01-01`,
    primary_release_date.lte: `${decade + 9}-12-31`,
    page,
    sort_by: 'popularity.desc',
  }));

Extend the Watchlist

The watchlist uses React Context + localStorage. To add features like ratings or notes, update src/context/WatchlistContext.js:

// Add a user rating when saving
const addWithRating = (movie, rating) => {
  setWatchlist(prev => [
    { ...movie, userRating: rating, addedAt: Date.now() },
    ...prev.filter(m => m.id !== movie.id)
  ]);
};

Troubleshooting
Common issues & fixes
❌ "Invalid API key" error / blank app

Your REACT_APP_TMDB_API_KEY is missing or wrong. Check that your .env file exists at the project root (same level as package.json), the variable name starts with REACT_APP_, and restart the dev server after any .env changes.

❌ "Cannot GET /movie/123" on refresh (Vercel)

Create a vercel.json file at the project root to handle client-side routing: 

{
  "rewrites": [{ "source": "/(.*)", "destination": "/" }]
}
❌ Environment variable not working in production

After adding/changing environment variables in the Vercel dashboard, you must redeploy the project. Go to Deployments → click the three dots on the latest build → Redeploy.

💡 Images not loading

TMDB images are served from image.tmdb.org. If images fail, your network may be blocking the CDN, or the movie simply has no poster (the app handles this gracefully with a placeholder).
